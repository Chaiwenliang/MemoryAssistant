设计说明

  • 双主题架构：Light（方案 A 极简白灰）和 Dark（方案 C 玻璃科技），两个主题共用同一套组件，仅通过 CSS class 切换配色
  • 字体与间距系统：标题 22px · 副标题 14px · 正文 13px · 小字 11px · 行间距 1.5-1.75，确保在两种主题下都有足够的阅读对比度
  • 输入框改进：placeholder 与图标之间有 10px 呼吸间距，底部输入条有左模式切换 / 中输入 / 右提交按钮三层结构，避免文字贴边
  • 圆角层级：卡片圆角 14px · 列表项无圆角（用分隔线）· 按钮圆角 10-14px · 芯片圆角 10px，层次清晰
  • 色板：Light 主色 #1c1c1e（深灰）+ #f2f2f7（浅灰） · Dark 主色 #6dd5fa（冰蓝）+ #ff9ec7（强调粉）
